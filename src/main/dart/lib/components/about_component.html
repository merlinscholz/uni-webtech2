About TuDo

TuDo is a simple ToDo-List-Webapp based on a Java EE-backend and REST-API, and an AngularDart-frontend. This webapp is a project for Webtech2 at the TU Dortmund.

© 2018 Vanessa Speeth & Merlin Scholz

Technologies used

  • Java Enterprise Edition 7
  • WildFly Application Server
  • Dart Programming Language
  • Apache Maven
  • Apache Shiro
  • H2 Database Engine
  • Hibernate ORM
  • JAX RS
  • Stanford Javascript Crypto Library
  • Linearicons Free
